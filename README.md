## Getting Started

- Clone the file to your computer
```bash
gh repo clone Seb-GRAF/antistaville
```

- Install dev dependencies:

```bash
npm install
```

- Run the local server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the site.
